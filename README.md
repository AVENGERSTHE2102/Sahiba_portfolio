# Sahiba Bali — Portfolio

> Official portfolio website for Sahiba Bali — Actor, Model & Content Creator based in Mumbai.

**Live:** [adityainfluencerportfolio.vercel.app](https://adityainfluencerportfolio.vercel.app)

---

## About

A high-performance, animation-rich portfolio built for Sahiba Bali showcasing her work as an actor (*Laila Majnu*, *Bard of Blood*, *Amar Singh Chamkila*), model, content creator, and former marketing professional at Zomato and Unacademy.

---

## Features

- **Cinematic loading screen** with animated letter reveals
- **Scroll-scrubbing animations** — images animate on scroll with parallax, scale, and color transitions
- **Bio image loop** — cycling portrait gallery with smooth transitions
- **Stagger hover effects** on all navigation links
- **Momentum scrolling** on desktop via Luxy.js
- **Fully responsive** — mobile-optimized layout with portrait hero image
- **SEO + AEO optimized** — structured data (Schema.org Person), Open Graph, Twitter Cards, sitemap, robots.txt

---
